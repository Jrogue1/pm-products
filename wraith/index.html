Questions

What AI systems can Wraith test?▼
Any AI system built on Claude, GPT, Gemini, Ollama, or custom LLM backends. Wraith tests the infrastructure around the model: prompt handling, memory systems, agent communication, voice pipelines, policy engines, and API surfaces. If it runs an AI agent, Wraith can scan it.
What's the difference between Base and Pro?▼
Both tiers include all 24 scanner modules. The difference is scope: Base is locked to localhost and private IP ranges (your own machine and local network). Pro unlocks external target scanning for testing remote infrastructure, plus includes a VPS deployment guide, custom module SDK, and priority support.
Does Wraith phone home?▼
No. License validation is entirely offline using cryptographic verification. No activation server, no telemetry, no usage tracking. Your scan results never leave your machine.
How does this compare to Burp Suite Pro?▼
Different tools for different jobs. Burp Suite is the industry standard for web application pentesting: HTTP traffic interception, manual request manipulation, browser-based testing. Wraith is purpose-built for AI system security: prompt injection, memory poisoning, agent exploitation, voice pipeline attacks. Burp costs $475 every year. Wraith is a one-time purchase. Most security professionals will want both.
Can I transfer my license to a new machine?▼
Yes. Contact support with your proof of purchase and we'll issue a replacement key. Each license allows up to 3 machine transfers.
What do I need to run Wraith?▼
Node.js 20+, npm, and a terminal. Optional: nmap and ffuf for network scanning modules (installable via package manager). Wraith runs on Linux, macOS, and WSL2 on Windows. No Docker required.
Is this legal to use?▼
Wraith is a security testing tool. Use it only on systems you own or have written authorization to test. The Pro tier requires explicit confirmation (--confirm-external flag) before scanning any external target. Unauthorized scanning of systems you don't own is illegal regardless of what tool you use.
